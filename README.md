# Brage

A react App implemented from all the steps in the tutorial: [A complete React with Apollo and GraphQL Tutorial](https://www.robinwieruch.de/react-graphql-apollo-tutorial).

BRAGE:  Brage (fn. Bragi) är skaldekonstens och vältalighetens gud. Berömd är han ock för sin visdom. Han kallas stundom för sitt långa skägg den sidskäggige åsen. Hans hustru heter Idun (fn. Iðunn) och hon är ungdomens och föryngringens gudinna. Hon förvarar i en ask de äpplen, af hvilka gudarne äta, när de åldras, och blifva de då åter alla unga.


  There are several different ways to implement and use
  fontawesome icons. I chosed to install the free packages
  and import the icons I wanna use in each file.
  See "Using Icons via Individual Use" here:
  https://fontawesome.com/how-to-use/on-the-web/using-with/react

## Installation

* `git clone git@github.com:Ojvind/brage.git`
* cd brage
* yarn install
* yarn start
* visit `http://localhost:3000`
